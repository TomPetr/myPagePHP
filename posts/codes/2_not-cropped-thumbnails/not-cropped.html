Not Cropped Thumbnails

Show Their Whole Beauty

On the art portfolio website, to have neat presentation of images of artwork is an essential task. And the thumbnail gallery will deliver it perfectly. All thumbnails create the grid of rows and columns, all thumbnails have the same sizes, and therefore the same width-to-height ratio. Everything is organized, easy to view and everything looks great.

But on the other hand, artwork isn’t as uniform, it shouldn’t be. The images can have various width-to-height ratio. If I want to insert a landscape-oriented image into square thumbnail frame, I have to set the CSS properties for this image like width: 100%; height: auto;, for example. But the next image may be portrait oriented one. Then CSS could be like width: auto; height: 100%;.

Most thumbnails galleries I could see crop the images to the same width-to-height ratio as their thumbnail frame. This can be done by static CSS. But it’s what I don’t want, I like to show my images as I created them, in full, not cropped. Then my audience can see them at a glance how they really are. I called javascript for the help to set CSS properties dynamically.
